# 👹 Guide des Malédictions (CURSES)

Le bot propose actuellement **21 types de malédictions** différentes que tu peux lancer sur les membres du serveur. 

### ⚙️ Syntaxe des Commandes
- **Lancer une malédiction aléatoire** : `!curse @joueur [durée]`
- **Lancer une malédiction spécifique (caché)** : `!curse hidden @joueur [durée] [TYPE]`
- **Exemple** : `!curse hidden @victim 15 UWU_MODE`

---

## 🧩 Nouveau : L'Épreuve Suprême
| Type | Nom | Effet |
| :--- | :--- | :--- |
| `CHALLENGE` | 🧩 **Épreuve du Scribe** | **Bloque tout (Chat + Vocal + Commandes)**. L'utilisateur doit recopier une phrase complexe (virelangue) pour être libéré. |

## 💬 Altération des Messages (Chat)
Ces malédictions modifient ce que l'utilisateur écrit dans les salons textuels.

| Type | Nom | Effet |
| :--- | :--- | :--- |
| `MESSAGE_SCRAMBLER` | 🔀 **Mélangeur de Mots** | Mélange l'ordre des mots dans les messages. |
| `MESSAGE_OPPOSER` | 🔄 **Opposé** | Dit l'inverse de ce que l'utilisateur voulait dire. |
| `CLOWN_MODE` | 🤡 **Mode Clown** | Ajoute des emojis 🤡 partout. |
| `UWU_MODE` | 😺 **Mode UwU** | Transformation en langage "UwU" (>///<). |
| `YODA_MODE` | 🗣️ **Mode Yoda** | Inverse l'ordre des mots façon Maître Yoda. |
| `CAPS_LOCK` | 📢 **CAPS LOCK** | TOUT EN MAJUSCULES AVEC DES POINTS D'EXCLAMATION!!!! |
| `PIRATE_MODE` | 🏴‍☠️ **Mode Pirate** | Ajoute des expressions de pirate (Arr, Moussaillon!). |
| `VOWEL_REMOVER` | 🎯 **Suppression Voyelles** | Retire toutes les voyelles des messages. |
| `REVERSE_TEXT` | 🔁 **Texte Inversé** | Écrit le texte à l'envers. |
| `RANDOM_EMOJI` | 🌈 **Emojis Aléatoires** | Remplace certains mots par des emojis bizarres. |
| `GARBLED` | 🔀 **Messages Déformés** | Déforme les commandes avant leur exécution. |
| `POLITE_MODE` | 🤵 **Mode Majordome** | Oblige à être poli (Monsieur... salutations distinguées). |
| `SHY_MODE` | 😳 **Mode Timide** | Limite les messages à 10 caractères et les tronque. |
| `EMOJI_ONLY` | 😜 **Mode Émoticône** | Seuls les emojis sont autorisés dans le chat. |
| `QUESTIONER` | ❓ **Point d'Interrogation** | Tous les messages doivent finir par un point d'interrogation. |
| `SELF_DESTRUCT` | 🧨 **Auto-Destruction** | Les messages se suppriment après 5 à 10 secondes. |
| `PARROT` | 🦜 **Le Perroquet** | Le bot répète tes messages de façon moqueuse. |

## 🛠️ Comportement du Bot & Commandes
Ces malédictions changent la façon dont le bot interagit avec l'utilisateur.

| Type | Nom | Effet |
| :--- | :--- | :--- |
| `RANDOM_RESPONSES` | 🎲 **Réponses Aléatoires** | Le bot répond n'importe quoi aux commandes. |
| `IGNORED` | 👻 **Ignoré** | Le bot ignore complètement l'utilisateur. |
| `BLOCKED` | 🚫 **Bloqué** | Aucune commande ne fonctionne pour l'utilisateur. |
| `WORST_LUCK` | 💀 **Malchance** | Tous les résultats aléatoires (dés, roulette) sont les pires possibles. |
| `SLOW_MODE` | 🐌 **Mode Lent** | Le bot met 10 secondes à répondre à l'utilisateur. |
| `REVERSED` | 🔄 **Commandes Inversées** | Les commandes de l'utilisateur se retournent contre lui. |
